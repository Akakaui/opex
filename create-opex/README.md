# @akakaui/create-opex

OPEX — AI Business Operating System for OpenCode & Claude Code

## Install

```bash
npx @akakaui/create-opex
```

## What It Does

Sets up a fleet of 15 specialized AI agents for:
- Content operations
- Client acquisition
- Sales pipeline
- Digital products
- Video production

## Interactive Setup

The installer will ask:
1. Agent name (default: opex)
2. Your business/brand
3. Main goal
4. Install for OpenCode, Claude Code, or both
5. API keys (optional)
6. Content platforms

## Platforms Supported

### OpenCode
- Agents installed to `.opex/agents/`
- Skills installed to `.opex/skills/`
- Use `@opex [request]`

### Claude Code
- Agents installed to `.claude/agents/`
- Config installed to `.claude/CLAUDE.md`
- Use `@opex [request]`

## What's Included

- 15 specialized agents
- 17 skill files
- Document generation tools
- YouTube transcription MCP server
- Configurable brand specs
